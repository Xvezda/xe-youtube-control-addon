xe-youtube-control-addon
========================
게시글에 삽입한 유튜브 비디오 플레이어의 옵션을 일괄적으로 조절할 수 있도록 도와주는 애드온입니다.

옵션조절 이외에도 비디오 플레이어 자동 리사이즈등의 편의기능을 포함하고 있습니다.
